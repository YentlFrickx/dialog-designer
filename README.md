# DnD AEM Dialogs 

Single Page Drag/Drop application to create AEM Dialogs.

Supports following fields:

###Layout

- tabs (section)
- fieldset

###Normal Fields

- textfield
- textarea
- numberfield
- richtext
- datepicker
- hidden

###Selection Fields

- select
- checkbox
- switch
- buttongroup
- radio

###Resource Fields

- fileupload
- pathbrowser
- pathfield

###Authoring Helpers

- anchorbutton
- alert
- heading
